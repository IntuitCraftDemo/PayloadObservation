# Craft Demo

## Description

It's the repository for intuit A4A craft demo interview. Requirements see [here](./SRE%20A4A%20Craft%20Demo%20(1).docx).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
